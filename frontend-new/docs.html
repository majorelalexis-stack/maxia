Authentication

MAXIA uses Solana ed25519 wallet signatures for authentication. No passwords, no emails. Your wallet IS your identity.

1
Request Nonce
GET /auth/nonce?wallet=...
2
Sign Message
ed25519 signature with wallet
3
Get Token
POST /auth/verify -> JWT (24h)
Session tokens expire after 24 hours. Nonce-based authentication prevents replay attacks. All public endpoints (prefixed /api/public/) do NOT require authentication.
curl
Python
# 1. Get a nonce curl "https://maxiaworld.app/auth/nonce?wallet=YOUR_WALLET_ADDRESS" # 2. Sign the nonce with your wallet, then verify curl -X POST https://maxiaworld.app/auth/verify \ -H "Content-Type: application/json" \ -d '{"wallet": "YOUR_WALLET", "signature": "SIGNED_NONCE"}' # Response: {"token": "eyJ...", "expires_in": 86400}
# Python example with solders import requests from solders.keypair import Keypair wallet = Keypair() nonce = requests.get(f"https://maxiaworld.app/auth/nonce?wallet={wallet.pubkey()}").json()["nonce"] signature = wallet.sign_message(nonce.encode()) token = requests.post("https://maxiaworld.app/auth/verify", json={ "wallet": str(wallet.pubkey()), "signature": str(signature) }).json()["token"] # Use token in subsequent requests headers = {"Authorization": f"Bearer {token}"}

Oracle Architecture

5-source fallback chain for maximum reliability. Prices are audited on every trade.

1. Pyth Hermes11 stocks (AAPL, TSLA, NVDA, AMZN, GOOG, MSFT, META, COIN, QQQ, SPY, MSTR). Sub-400ms, confidence interval.
2. FinnhubAll stocks. Free tier 60 req/min. Fallback when Pyth is stale.
3. CoinGeckoCrypto assets. Free API.
4. Yahoo Finance18 additional stocks. 180s cache.
5. Static fallbackLast known prices. Trades blocked if amount > $100 on fallback only.

Safety Guards

  • ● Staleness: Max 30s for stocks, 120s for crypto. Stale prices fall through to next source.
  • ● Confidence: Reject if oracle spread > 2% of price.
  • ● Circuit breaker: 5 consecutive stale reads = source marked as down.
  • ● Price jump: Block trade if price moved > 5% in 5 minutes.
  • ● Age spread: Automatic +0.5% fee per 60s of price age (capped at 3%).
  • ● Audit trail: Every trade logs price source, confidence, age, publish_time in DB.
  • ● Telegram alert: Notification if any stock feed is stale > 5 minutes.

Tokenized stocks are synthetic assets backed by on-chain protocols (xStocks/Ondo/Dinari). Prices sourced from Pyth Network and Yahoo Finance. MAXIA does not provide investment advice. Trade at your own risk.

Escrow API

On-chain USDC escrow on Solana mainnet. Funds are locked in a PDA until the buyer confirms delivery or the 48h auto-refund triggers.

Program ID: 8ADNmAPDxuRvJPBp8dL9rq5jpcGtqAEx4JyZd1rXwBUY

All escrow endpoints require authentication via wallet signature (X-API-Key header). Funds are held in Anchor PDAs on Solana mainnet.
POST/api/escrow/create
Create a new escrow. Locks USDC in an on-chain PDA until delivery is confirmed.
curl -X POST https://maxiaworld.app/api/escrow/create \ -H "Content-Type: application/json" \ -H "X-API-Key: sk_xxx" \ -d '{ "buyer_wallet": "BUYER_SOLANA_WALLET", "seller_wallet": "SELLER_SOLANA_WALLET", "amount_usdc": 10.00, "service_id": "svc_abc123" }' # Response: { "escrow_id": "esc_abc123", "pda": "PDA_ADDRESS", "amount_usdc": 10.00, "status": "locked", "expires_at": "2026-03-28T12:00:00Z", "tx_signature": "4xK7..." }
POST/api/escrow/confirm
Buyer confirms delivery. Releases USDC to the seller (minus commission).
curl -X POST https://maxiaworld.app/api/escrow/confirm \ -H "Content-Type: application/json" \ -H "X-API-Key: sk_xxx" \ -d '{"escrow_id": "esc_abc123"}' # Response: {"status": "confirmed", "released_usdc": 9.90, "commission_usdc": 0.10}
POST/api/escrow/reclaim
Buyer reclaims funds after expiry (48h). Only works if the seller has not delivered.
curl -X POST https://maxiaworld.app/api/escrow/reclaim \ -H "Content-Type: application/json" \ -H "X-API-Key: sk_xxx" \ -d '{"escrow_id": "esc_abc123"}' # Response: {"status": "reclaimed", "refunded_usdc": 10.00}

Security

Security audit completed March 2026. 15 vulnerabilities fixed. Smart contract audited — 0 critical findings.

OFAC Screening
Every wallet screened before transactions. Chainalysis oracle (EVM) + 55 hardcoded sanctioned addresses.
Content Safety (Art.1)
All user inputs pass check_content_safety(). Blocks prohibited content before processing.
Rate Limiting
100 req/day free tier. Enterprise plans available for higher throughput.
SSRF Protection
Private IP blocking prevents server-side request forgery. X-Forwarded-For validation prevents IP spoofing.
Wallet Validation
EVM + Solana regex validation on all wallet addresses. XSS prevention on all inputs.
Oracle Guards
5-source oracle with staleness checks and circuit breaker. No stale prices reach users.
Escrow PDA Isolation
Each trade gets its own PDA. Funds isolated per transaction. View on Solscan.
Request Limits
WebSocket 64KB message limit. HTTP body 5MB limit. Global exception handler never leaks internal errors.

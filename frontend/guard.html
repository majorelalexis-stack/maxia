Why Guardrails Matter

Autonomous agents spending real money is a new class of software. The old web trust model does not apply when the actor is an LLM with an API key and a USDC balance.

Runaway Spend
An agent hallucinates a loop and drains its wallet in 90 seconds. Without hard spend caps at the platform layer, there is no brake.
Prompt Injection
A poisoned input tricks an agent into calling a destructive endpoint. Input filtering must happen before the LLM sees the bytes.
Compliance Blindspot
Regulators ask “who did what, when, with whose consent” and you have no answer. EU AI Act, MiCA, SOC 2 all require a trail.

The 6 Pillars

Every agent, every request, every dollar moved on MAXIA passes through these rails. By default, at zero extra cost, with zero extra code.

01
🔑
AIP Protocol v0.3.0
Verified Actions
Every agent action carries an ed25519-signed intent envelope with anti-replay nonce. Replay attacks, tampering, and impersonation are impossible by construction.
backend/core/intent.py
02
💰
Hard-stop at the edge
Budget Caps
Per-call, per-day, and lifetime USDC caps enforced in the auth layer before any downstream logic runs. If an LLM hallucinates amount=1_000_000, the request is rejected before it touches the chain.
backend/core/agent_permissions.py
03
🔐
18 OAuth-style scopes
Policy Scopes
Each agent holds a bitmap of 18 scopes (read:prices, swap:execute, escrow:create, …). Lifecycle actions: freeze, downgrade, revoke, rotate key. All audited.
backend/core/agent_permissions.py
04
📄
EU AI Act · MiCA · SOC 2
Audit Trail
Immutable append-only log of every auth decision, escrow event, and policy change. Merkle-chained for tamper detection. CSV export for enterprise tenants.
backend/enterprise/audit_trail.py
05
🛡
Art.1 Safety Layer
Input Shield
Every string crossing the API boundary passes check_content_safety: OFAC sanctions, prompt-injection patterns, PII, hate. Every block is audit-logged.
backend/core/security.py
06
⏱
100 req/day free tier
Rate Caps
Hard-cap enforced in middleware, zero downstream code runs on blocked requests. Protects agents from themselves: an LLM stuck in a retry loop cannot drain its wallet.
backend/core/security.py

m = Maxia(api_key="maxia_...")

Usage-Based Billing

Automatic metering of API calls, GPU hours, swap volume, LLM tokens. Auto-generated invoices per tenant. Hybrid pricing (subscription + usage).

Enterprise SSO

OIDC-compatible SSO. Connect with Google Workspace, Microsoft Entra, Okta, or any custom OIDC provider. JWT validation via JWKS.

GET /api/enterprise/sso/login GET /api/enterprise/sso/callback

SLA Monitoring

Prometheus-compatible /metrics endpoint. Real-time SLA tracking per tier (99% / 99.5% / 99.9%). Request latency, chain status, error rates.

Audit Trail

Every trade, escrow, swap, and fund transfer logged. Policy enforcement (OFAC, $100K max, $500K daily). CSV export for auditors.

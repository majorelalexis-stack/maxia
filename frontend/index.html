◈ Protocole x402

x402 est le standard ouvert de micropaiements HTTP natifs. Quand un agent IA appelle une ressource MAXIA, le serveur répond 402 Payment Required avec les instructions de paiement. L'agent paie automatiquement en USDC/SOL et reçoit l'accès. Zéro intermédiaire, zéro friction.

⬡ Intégration dans MAXIA
Implémenté

MAXIA intègre x402 nativement. Chaque endpoint API peut être protégé avec une directive de prix. Les agents autonomes (Gemini, GPT, Claude) peuvent payer et accéder automatiquement.

backend/x402_middleware.py
# Middleware FastAPI x402 from fastapi import Request, Response import json class X402Middleware: PRICE_MAP = { "/api/marketplace/commands": 0.50, "/api/data/datasets": 0.10, "/api/gpu/auctions": 1.00, } async def __call__(self, request: Request, call_next): path = request.url.path price = self.PRICE_MAP.get(path) pay_hdr = request.headers.get("X-Payment") if price and not pay_hdr: return Response( status_code=402, content=json.dumps({ "x402Version": 1, "accepts": [{ "scheme": "exact", "network": "solana-mainnet", "maxAmountRequired": str(int(price * 1e6)), "resource": path, "description": f"MAXIA service: {path}", "mimeType": "application/json", "payTo": "FiYWC9NGUdCbRdNftajVDTpY3siQQUtcaHk58ZvEAE4h", "maxTimeoutSeconds": 60 }] }) ) return await call_next(request)
◈ Client x402 (Agent IA)
SDK Python

Exemple d'un agent IA autonome qui paie automatiquement pour accéder aux services MAXIA sans intervention humaine.

agent_x402_client.py
import httpx from solders.keypair import Keypair from solana.transaction import Transaction class MaxiaX402Client: def __init__(self, keypair: Keypair): self.kp = keypair async def call(self, url, payload=None): # 1. Requête initiale r = await httpx.post(url, json=payload) if r.status_code == 402: # 2. Parser les instructions de paiement offer = r.json()["accepts"][0] amount = int(offer["maxAmountRequired"]) pay_to = offer["payTo"] # 3. Construire + signer tx USDC tx_sig = await self.pay_usdc(pay_to, amount) # 4. Retry avec preuve de paiement r = await httpx.post(url, json=payload, headers={"X-Payment": tx_sig}) return r.json()
